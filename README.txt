You can dowload this project and run it on your own local server (option 1.) or you can view functionality of this project live (option 2.).

Option 1.
You can use XAMPP to simulate live web applications.
1. Download the project from git hub.
2. Create pwa_pravi folder in htdocs and extract all files into that folder.
3. Enter you phpmyadmin
4. click on new database
5. select import from the bar on the top of the screen
6. use sql script from the dowloaded project in db/pwa_database.sql
7. import it.

application is now ready to use.
